    Chatterboxes

    Chatterboxes

    Demo GitHub

    An augmentative and alternative communication (AAC) tool, giving a voice to those who may not have one. Search for symbols and pictures, create and edit your collection of symbols and build sentences that can be spoken out loud by speech synthesis. Chatterboxes can be used as a toolfor speech modeling, sentence writing,or used to reinforce meaning for those who have English as an additional language. As Chatterboxes is a progressive Web App, it can be easily downloaded to a mobile device, free of charge, and uses open source symbols making this an accessible tool for those who need it.

    • React Frontend, Ruby on Rails Backend
    • Hosted on Firebase and Heroku
    • Progressive Web App
    • Uses Web Audio API and OpenSymbols API

    Complimenter

    Complimenter

    Demo GitHub
